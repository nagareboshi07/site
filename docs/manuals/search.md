# ニコニコ検索シート(ver.2.xx) マニュアル
## 全体の変更点
- コンテンツ検索APIの対象サービス削減に対応。
- 検索対象サービスにより、検索シートが別のものに分けた。
  ver.2.00で対応予定のサービスは以下の通り。
  - ニコニコ動画(ニコニココンテンツ検索API、スナップショット検索API) 
  - ニコニコ生放送(ニコニココンテンツ検索API) 
  - ニコニコ静画(HTML解析)
- シート全体のレイアウト調整
- 内部的な変更
  - ニコニコのサーバに負担を与えないよう、動作時間の制限、あるいは待機時間を設けるように変更。
  - コード全体をリファクタリングした。
